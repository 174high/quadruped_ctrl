# quadruped_robot

### MIT mini cheetah simulation in pybullet

### System requirements:
Ubuntu 19.04, ROS Mellodic  

### Dependency:
```
git clone https://github.com/Derek-TH-Wang/cheetah.git
```

### Running:
```
rosrun cheetah QuadListener.py
roslaunch quadruped_robot quadruped_robot.launch
```
